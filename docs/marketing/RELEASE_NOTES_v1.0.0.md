# Music Saver v1.0.0

Первый публичный релиз desktop-приложения для переноса музыки из VK и Яндекс Музыки в локальную библиотеку на Windows.

## Что внутри

- Готовая Windows-сборка в GitHub Releases (`MusicSaver-windows.zip`).
- GUI на CustomTkinter для повседневного сценария "вставил ссылку -> скачал".
- Скачивание и обработка музыкальных файлов с автозаполнением ID3-тегов.
- Работа с обложками и сохранением структуры локальной медиатеки.
- История загрузок и настройки в локальном профиле.

## Для кого

- Для пользователей с крупными плейлистами и архивами (500+ треков).
- Для тех, кто хочет офлайн-доступ и контроль над своей библиотекой.

## Важно

Интеграции с VK/Яндекс зависят от текущей вёрстки и API платформ. Если после обновления сервисов что-то перестало работать, создайте Issue со шагами воспроизведения и логами: выпустим hotfix.
